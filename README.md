
# LangChain Learning Projects

This repository contains a collection of small projects and experiments built while learning **LangChain**, AI agents, and LLM-based automation systems.

The goal of this repository is to explore how **AI agents, search tools, and conversational systems** can be built using LangChain and Python.

## 🚀 Projects Included

### 1️⃣ Google Search Agent
An AI agent that can perform **web searches and return intelligent answers** using LangChain.

- Processes queries through an LLM
- Returns summarized answers

```

apps/Google_search_agent_debug.py

```

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/langchain-learning.git
cd langchain-learning
````

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Setup

Create a `.env` file and add your API keys.

Example:

```
OPENAI_API_KEY=your_api_key_here
```

You can add any other API keys depending on the tools used in the project.

---

## ▶️ Running the Projects

Run any project file using Python.

Example:

```bash
python apps/QnA_chatbot_app.py
```

Or run the search agent:

```bash
python apps/Google_search_agent.py
```
